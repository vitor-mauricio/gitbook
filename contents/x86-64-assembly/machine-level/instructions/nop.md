# NOP

A instrução **NOP** (“No Operation”) em x86-64 é uma das instruções mais simples do conjunto — ela literalmente **não faz nada observável** além de consumir tempo (1 ciclo, dependendo do processador) e avançar o ponteiro de instruções (**RIP**) para a próxima instrução.

## Syntax

```asm
nop
```

É uma instrução sem operandos.

## Examples

```asm
nop
```
