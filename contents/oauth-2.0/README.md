# OAuth 2.0

## Motivação

Devido a algumas iniciativas no meu trabalho decidi estudar um pouco melhor sobre o funcionamento do protocolo, principalmente sob o ponto de vista de segurança. O estudo teve 2 motivações simples:

* O funcionamento do protocolo;
* Como utilizá-lo da melhor forma - garantir a segurança.
